{# 模板语言的注释 #}

1.变量:

name:{{ name }}

age:{{ age }}

likes:{{ likes }}

2.标签:

if语句
{% if age >= 18 %}

{{ name }}已经成年了

{% elif age >= 6 %}

{{ name }}可以上学了

{% else %}

{{ name }}还是小孩啊

{% endif %}

3.for 循环

{% for like in likes %} {% if loop.first %}

{{ like }}

{% elif loop.last %}

{{ like }}

{% else %}

{{ like }}

{% endif %} index:{{ loop.index }}, index0:{{ loop.index0 }}, revindex:{{ loop.revindex }}, revindex0:{{ loop.revindex0 }} {% else %}

如果循环出现问题 则进行到这个else语句

{% endfor %}
